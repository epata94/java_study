<String 클래스 연습문제>
1. 이름, 국어, 영어, 수학 점수를 한번에 입력받아 평균점수를 출력하는 예제 코드
- String클래스의 split함수를 활용하세요
* 화면 출력
이름과 국어, 영어, 수학 점수를 입력하세요(예: 홍길동 90 85 95): 홍길동 90 85 95
이름: 홍길동
국어 점수: 90
영어 점수: 85
수학 점수: 95
평균 점수: 90.0

2. String의 contain() 메소드를 사용하여 문자열에 특정 문자열이 포함되어 있는지를 확인하세요.
 - 아래와 같이 문자열 및 검색문자열을 지정하세요.
        String str = "나는 자바 프로그래밍을 배우고 있습니다.";
        String searchStr = "프로그래밍";
 - 포함되어 있지 않다면
 문자열에 '[검색어]'이(가) 포함되어 있지 않습니다.
* 화면 출력
문자열에 '프로그래밍'이(가) 포함되어 있습니다.

3. String의 indexOf() 메소드를 사용하여 문자열에서 특정 문자열의 인덱스를 확인합니다.
 - 아래와 같이 문자열 및 검색문자열을 지정하세요.
        String str = "나는 자바 프로그래밍을 배우고 있습니다.";
        String searchStr = "프로그래밍";
 - 포함되어 있지 않다면
문자열에 '[검색어]'이(가) 포함되어 있지 않습니다.
* 화면 출력
문자열에서 '프로그래밍'이(가) 위치한 인덱스: 6

<String Buffer 연습문제>
4. StringBuffer 클래스를 사용하여 입력받은 한글 문자열을 거꾸로 출력하는 예시 코드입니다.
- StringBuffer 클래스에서 r로 시작하는 메소드를 찾아보세요.

* 화면 출력
문자열을 입력하세요: 홍길동
입력한 문자열의 역순은 다음과 같습니다: 동길홍

5. StringBuffer 클래스를 사용하여 입력받은 한글 문자열의 앞뒤 공백을 모두 제거하세요.
- String클래스의 trim 함수를 사용하세요.
* 화면 출력
문자열을 입력하세요:    으라차차!    스프링! 기다려~~~~
앞뒤 공백이 제거된 문자열은 다음과 같습니다: 으라차차!    스프링! 기다려~~~~

6. StringBuffer 입력받은 한글 문자열에서 특정 문자를 모두 제거하는 예시 코드입니다.
- while문을 사용하세요.
- StringBuffer의 indexOf, deleteCharAt 메소드를 사용하세요.

문자열을 입력하세요: 나는 일개미입니다. 일개미는 바쁩니다.
제거할 문자를 입력하세요: 일
제거된 문자열은 다음과 같습니다: 나는 개미입니다. 개미는 바쁩니다.
